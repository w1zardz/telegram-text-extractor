# Telegram Text Extractor — копировать текст из закрытых Telegram-каналов

> **Расширение для браузера, которое позволяет скопировать текст из любого Telegram-канала или группы в Telegram Web — даже если автор канала включил «Запретить сохранение контента».** Поддерживает все версии `web.telegram.org` (K, A, Z и legacy). Копирование одного поста, всех постов сразу, экспорт в `.txt`. **Manifest V3.** Никакие данные не уходят с компьютера.

## Что решает

В Telegram есть настройка канала **«Запретить сохранение контента»** (`Restrict saving content`). Когда она включена:

- в канале нельзя выделить текст,
- из контекстного меню пропадает пункт «Копировать»,
- `Ctrl+C` / `Cmd+C` не работают,
- сообщения нельзя пересылать в другие чаты.

Запрет реализован **на стороне клиента** в UI Telegram Web — текст всё ещё лежит в DOM страницы, просто интерфейс не даёт его выделить. Расширение читает текст напрямую из DOM, выводит его в собственную панель (где выделение не заблокировано) и даёт нормальную кнопку «Скопировать».

## Возможности

- 📋 Копирование текста из защищённых от копирования каналов и групп
- 📜 Массовое копирование всех видимых сообщений одной кнопкой
- 💾 Экспорт в `.txt` с временной меткой в имени файла, UTF-8
- 🎯 Кнопка «→ Jump» — перейти к исходному сообщению в чате

- 🛡️ 100% локально — никаких сетевых запросов, аналитики, телеметрии, удалённой конфигурации
- 🪶 Без зависимостей и без сборки — vanilla JS, ~14 КБ
- 🌗 Тёмная панель, совместимая с темой Telegram
- ⚡ Manifest V3 — Chrome / Edge / Brave / Opera / Vivaldi / Arc

## Установка

1. Скачай репозиторий: `git clone https://github.com/w1zardz/telegram-text-extractor.git` (или ZIP-ом).
2. Открой `chrome://extensions` (или `edge://extensions`, `brave://extensions`).
3. Включи **Режим разработчика** в правом верхнем углу.
4. Жми **Загрузить распакованное расширение** и выбери папку `telegram-text-extractor`.
5. Открой [web.telegram.org](https://web.telegram.org) — внизу справа появится круглая кнопка **📋**.

### Firefox

Для Firefox пока используй userscript-фолбэк: [`docs/userscript.user.js`](docs/userscript.user.js) с Tampermonkey или Violentmonkey.

## Как пользоваться

1. Открой нужный канал/группу на `web.telegram.org`.
2. Прокрути до постов, которые нужно скопировать (Telegram рендерит только видимые сообщения — старые подгружаются по мере скролла).
3. Нажми **📋** в правом нижнем углу — выедет панель.
4. Жми **↻** для сканирования. Дальше:
   - **📋 Copy** — скопировать один пост,
   - **Copy all** — скопировать всё видимое,
   - **⬇ .txt** — скачать файлом `tg-posts-YYYY-MM-DD-HH-MM.txt`.

Чтобы выгрузить длинный канал — медленно скролль вверх с открытой панелью, расширение само добавляет новые посты по мере их рендера.

## FAQ

### Работает ли в каналах с включённым «Запретить сохранение контента»?
Да, именно для этого расширение и сделано.

### Работает ли в десктопном/мобильном Telegram?
Нет. Десктоп и мобильный клиенты рендерят текст в нативном коде, а не в DOM. Открой Telegram **Web** в браузере: `web.telegram.org`.

### Узнает ли админ канала, что я скопировал пост?
Нет. Всё происходит локально. Сетевых запросов расширение не делает — это можно проверить в DevTools → Network.

### Можно ли скачать фото / видео / голосовые?
Нет, расширение работает только с текстом.

### Это нарушает ToS Telegram?
Расширение не обращается к API Telegram. Оно читает DOM страницы, которая уже загружена в твоём браузере — те же данные, которые видят твои глаза. При этом автор канала специально просил не распространять его контент — уважай это. Инструмент предназначен для личного использования: заметок, архивации, доступности.
